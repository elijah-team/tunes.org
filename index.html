The TUNES OS/Language Project

Project Status: INACTIVE

TUNES started in 1992-95 as an operating system project, but was never clearly defined, and it succumbed to design-by-committee syndrome and gradually failed. Compared to typical OS projects it had very ambitious goals, which you may find interesting. However, we must warn you that our website is disorganized.

There is a marginally active TUNES Project Google Group, with 20 members as of April 2011. Think of it as a computer visionaries' club.
