.NET VM

A concrete VM inspired by a Microsoft proprietary VM (.Net) with an open published standard (ECMA 335, see below); it provides a means for loosely combining languages through a "Common Language Infrastructure" (CLI) in which applications written in multiple high-level languages may be executed in different system environments without porting.

  • ECMA 335.
