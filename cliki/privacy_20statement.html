Privacy Statement

In normal operation, the Tunes project's CLiki and the webserver it runs on will collect information about the people browsing and contributing to it.

This information includes:

  • Server logs collected by apache (see Combined Log Format on the Apache site).
  • Text that you provide when editing or adding pages to this web site. This is - obviously - not private. Please note that no authentication of users is done yet, so there is no sure way to identify who added what to which page.
