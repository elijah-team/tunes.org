Alice

A functional programming language extended from Standard ML with support for concurrent, distributed, and constraints programming. It is implemented on the Mozart/Oz platform.

The features distinguishing it from ML are:

  • Laziness
  • Futures
  • Higher-Order Modules
  • Components
  • Pickling-based persistence
  • Constraints
