Silk

SILK (Scout in Linux Kernel) is a Linux kernel module which transforms a standard Linux OS into Scout.

It appears that there is a fully functioning implementation which is part of PlanetLab (an open, globally distributed testbed for developing, deploying and accessing planetary-scale network services). Both come from the University of Princeton Network Systems Group, with SILK public release not available in early 2003.
