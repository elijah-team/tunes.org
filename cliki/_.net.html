.NET

.NET (pronounced "dot net") is a marketing name assigned to a range of Microsoft technologies and products. It most commonly refers to the extensive .NET class hierarchy, the core .NET languages supplied by Microsoft (C++, VB, C#, and others), the runtime system and it's related underpinnings (CLI, CLR, IL), the component and type systems, and so on.
