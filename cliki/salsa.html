Salsa

An acronym for Simple Actor Language and System Architecture: a distributed actor-based programming language with syntax similar to Java's, while also running on the JVM platform. It is intended to run alongside a World-Wide Computing (WWC) architecture which provides naming services and messaging.

  • SALSA Programming Language.
