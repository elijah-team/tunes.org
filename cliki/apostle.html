Apostle

Apostle OS project, initiated early october 1999, and managed by Guido de Jong. The page has not been updated since march 2001. The project is based on a microkernel, for which has been started a project apart, Gemini Nucleus, whose inspiration sources are L4, Aegis and EROS. C++ was considered as the implementation language for a while but the team reverted back to the C language at the first sight of bloat.
