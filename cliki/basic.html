BASIC

A Programming Language designed at Dartmouth College in 1964 to teach introductory programming. The name is an acronym for "Beginner's All-Purpose Symbolic Instruction Code". Among programming languages, BASIC is considered easy to learn. At first the language was not considered useful for real-world applications, but newer implementations expand greatly upon BASIC's original limited functionality. It is usually interpreted.
