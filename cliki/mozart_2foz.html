Mozart/Oz

A two-layered system, consisting of:
  • Mozart, an open distributed system providing constraints-based inference.
  • Oz, a concurrent object-oriented programming language implemented in Mozart.

This combination allows for multiparadigm programming, essentially. It is apparently equivalent in expressive power to aspect-oriented programming. The sole implementation is a cross-platform VM and run-time, with integration into the Tk graphics library and numerous other features.

See also PL 101 for an on-line book on programming that uses Mozart (Concepts, Techniques and Models of Computer Programming).
