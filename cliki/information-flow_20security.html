Information-Flow Security

A security concept concerned with controlling the flow of information within a system. In particular, it expresses information flow from endpoint to endpoint rather than step-wise as a capability system generally does. It does not encompass encryption-based security or access-control lists, but rather expresses something alongside them.

  • See Language-Based Information-Flow Security by Andrew C. Meyers at Cornell University, a recent paper describing the language-based security effort in this field with a good survey, an overview, and an excellent bibliography.
