KML

Kernel Mode Linux is a Linux OS variant which execute user processes in kernel mode.

In conjunction with TAL it could provide a no-kernel system. See Toshiyuki Maeda master thesis "Safe execution of user programs in kernel mode using Typed Assembly Language" ( .pdf, .ps ).
