Component Object Model

A Microsoft component system (see Combining Languages) which promoted strongly-typed contractual interfaces, interface and class versioning, distributed programming (Distributed COM -- DCOM). It has since been replaced by the component model in .NET system. COM was derived from an older component system known as OLE.
