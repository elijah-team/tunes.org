Plan9

Plan9 OS, or what Unix should have been (originally by Bell Labs from where Unix came). It's open source.

It got everything right as for proposing a uniform system interface (the dynamic, per-process name->channel "file system" thing) but is still wrong by using a low-level programming language with coarse-grained system abstractions.

See its successor, Inferno.
