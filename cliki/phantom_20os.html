Phantom OS

Balazs Miklos's Phantom OS MIA is a project for an OS for x86 PCs, based on a microkernel, and released under the GNU General Public License. Currently the kernel is being developed, but the plan is to make a full OS around it as time passes. There is sparse documentation but also some actual code to download.
