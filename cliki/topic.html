Topic

This is a list of the generally-intended topics for the Tunes CLiki.

You might say that it's the topic of topics. It's rather meta-circular, actually. :-)

Of course, you can use or add any topic you want.

  • Abstract VM - This is the topic of machines whose form is chosen so that it's easier to know certain properties of programs written for them
  • Acronym - The topic of Glossary acronyms
  • Alias - The topic of aliases for persons in the TUNES project
  • C language - The most common modern systems programming language to date, by Brian Kernighan and Dennis M. Ritchie (K&R); also the topic for the C language family
  • Combining Languages - The topic for the notion of combining the features, syntax, or semantics of several programming languages into one system
  • Comment - The topic of separate comment nodes about other nodes: statements which are too big to append to another page, but not long enough to consider a full essay or a formal proposal
  • Computer Algebra System - Uncategorized: the topic for systems which provide the means to deal with mathematical concepts computationally at various levels of abstraction
  • Concrete VM - This is the topic for machines with concrete implementation and precise byte-level encoding, that you can readily use to transmit code
  • Contributor - The topic of persons who have contributed in some way to this CLiki or to the main Tunes project
  • CPU - The topic CPU is an acronym for Central Processing Unit
  • Delete Me - The topic of nodes that will be deleted
  • Dispatch - The term and topic of a programming language's method of choosing a definition to use to evaluate some expression
  • Educational - The topic of projects which exist solely for educational purposes
  • Essay - Here are some rants, essays, and political perspectives on TUNES
  • Example - The topic of providing examples for guiding both the Tunes specifications and for illustrating and contrasting for newcomers the way Tunes is intended to behave
  • License - A software license is a set of terms which specify how the user can use, modify or distribute the software
  • OS - The topic OS is an acronym for Operating System
  • Person - The topic of people cited on this Wiki:
  • Political Term - Political terms in the Tunes lexicon
  • Program Transformation - The topic of metaprogramming techniques that act on whole programs, which also encompasses code generation and optimization
  • Programming Language - A topic for programming languages, a term for computational languages that are at least as expressive as the lambda calculus
  • Proprietary VM - This is the topic for machines you shouldn't use, least you'll become the hostage of the vendors who lock you and whoever else you will communicate with you into their technology
  • Resource - The topic of resources that are useful for building up the systems and features that interests TUNES, but don't fit into the Review areas
  • SubProject - The topic of sub-projects of the overall TUNES Project
  • Syntax - The term and topic of the structure of expressions in a language
  • Term - The topic of Glossary terms
  • Topic - This is a list of the generally-intended topics for the Tunes CLiki
  • User Interface - The topic of all user interfaces
  • VM - An acronym for Virtual Machine

For a way of defining and interchanging topics in the form of "topic maps" (ISO 13250) see the resources listed here, and the book "Topic Maps" from Addison-Wesley (available from Amazon).
