Libero

Libero is a metaprogramming tool to generate programs skeletons from state diagrams (FSM). Current programming languages supported includes C, C++, Java, REXX, Awk, Perl, Cobol, 80x86 assembly, etc. You should also take a look at MetaL if you prefer an XML high-level language to state diagrams.
