XL

The programming language for the Mozart framework. From What is XL? (see below):

XL is a new state of the art programming language. It addresses modern problems, using the ideas of concept programming. In particular, XL is designed to grow with Moore's law, since the programmers' brains don't grow that fast, but the programs complexity still does.
