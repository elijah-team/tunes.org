Resource

The topic of resources that are useful for building up the systems and features that interests TUNES, but don't fit into the Review areas.

  • ccg - By Ian Piumarta's, a resource for dynamic code generation in C and C++
  • Chord - A resource pertaining migration
  • Clump - A program that compiles and links C source files portably, without makefiles (a resource)
  • DCC Decompiler - A resource which is a proof-of-concept for decompilation of binary programs
  • DERIVE - A tool (resource) to reverse-engineering instruction encodings
  • DiskSim - DiskSim is a resource, The DiskSim Simulation Environment
  • Free Compilers Catalog - A resource of free compilers and interpreters (also by FTP) of programming languages
  • Graphics Standard - A resource for established Graphics Standards:
  • Memory-Pool System - A resource: a client-server architecture for allowing multiple memory-allocation strategies to co-exist for the same environment by partitioning areas into pools
  • Miasma - A resource, a machine-code toolkit for the x86 written in Scheme
  • New Jersey Machine Code Toolkit - A resource in ML for helping programmers write assembly-related applications that process machine code: assemblers, disassemblers, code generators, tracers, profilers, and debuggers
  • Nexus - A resource, a Multithreaded Communication Library
  • Obje - A framework resource to provide ubiquitous interoperability between digital devices
  • OLIVE - A collection of resources related to information visualization (HCI, User Interface)
  • On Automatic Interface Generation - See Tunes Interfaces Subproject -- Interface Generation
  • OSKit - A complete modular infrastructure that provides all the "bricks" (resources) needed to easily build an OS, focusing on what interests you, and reusing components for the rest
  • Planning Resources - AI Planning Resources:
  • Prevayler - A resource for object serialization and persistence
  • Psychology of Programming - An HCI resource about psychological aspects of programming and computational aspects of psychology
  • Systems Code Resources - Resources for low-level system services, including OS facilities and other libraries:
  • Zero Memory Widget - A user interface resource, a widget library that uses zero bytes of memory per widget
