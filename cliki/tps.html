Tps

An acronym for Tiny/Transportable Postscript, a Postscript-like programming language with all of the graphics operators removed and Heterogeneous State Transportability: the ability to interrupt-migrate-resume a computation (even on a different machine architecture/OS) conceived to programming Agents.
