Oz

Oz is a multiparadigm programming language that integrates in a clean way many useful programming concepts. Oz has a simple formal semantics and an efficient implementation (Mozart Programming System). Oz is used in the textbook Concepts, Techniques, and Models of Computer Programming [Note that this book is a suggested reading in our PL 101 -- MaD70].
-- Peter Van Roy
