Acronym

The topic of Glossary acronyms.

  • AI - An acronym for "Artificial Intelligence"
  • aka - An acronym for "also known as"
  • AOP - An acronym for "Aspect-Oriented Programming"
  • API - An acronym for Application Programming Interface
  • BNF - An acronym for Backus Normal Form or Backus Naur Form, a notation for context-free grammars
  • BOS - An acronym for Bubach Operating System, apparently a no-kernel OS
  • CAM - CAM is an acronym of Categorical Abstract Machine
  • CAPE - An acronym for Clips And Perl with Extensions: a programming environment which combine (see Combining Languages) CLIPS and PERL
  • CAS - An acronym for Computer Algebra Systems
  • CLHS - Search Facilities
  • CLI - An acronym for Command Line Interface, a kind of user interface
  • CLIM - CLIM is an acronym for Common Lisp Interface Manager
  • CLIPS - An acronym for C Language Integrated Production System; a programming language and an environment to development expert systems with three different programming paradigms: rule-based (declarative), object-oriented and procedural (imperative)
  • Cobol - An acronym for COmmon Business-Oriented Language, a very old programming language
  • COM - An acronym for Component Object Model
  • CORBA - An acronym for Common Object Request Broker Architecture
  • CPS - CPS is an acronym for Continuation-Passing Style
  • CPU - The topic CPU is an acronym for Central Processing Unit
  • CTO - An acronym for the site name CLiki.Tunes.Org, this site
  • DAG - An acronym for Directed Acyclic Graph, a graph with directed edges and no loops
  • DBMS - An acronym for Data Base Management System
  • DML - An acronym for Dependent ML, a conservative extension of the functional programming language ML with a type system enriched with a restricted form of dependent types
  • DSL - An acronym for Domain-Specific Language
  • DWIM - An acronym for a user interface strategy called "Do What I Mean", where common programming errors are corrected in a live sense
  • EBNF - An acronym for Extended BNF, a notation for context-free grammars
  • FAQ - An acronym for a list of Frequently-Asked Questions
  • FCCM - FCCM is an acronym for field-programmable custom computing machine, a computing device in which Programmable Logic Devices replace traditional general purpose processors, or fixed logic, in order to achieve superior performance or other capabilities
  • FFI - FFI is an acronym for Foreign Function Interface
  • FGCS - An acronym for Fifth Generation Computer Systems
  • FOP - An acronym for "Feature-Oriented Programming"
  • FSM - An acronym for Finite-State Machine
  • GC - An acronym for Garbage Collection
  • Godiva - An acronym for GOal-DIrected jaVA, a programming language which extends Java whith additional built-in data types, higher level (collection-oriented) operators, goal-directed expression evaluation (as in Icon), and pattern-matching on strings
  • GOMS - A user interface concept, acronym for Goals, Operators, Methods and Selection rules, a model of efficiency in HCI
  • GPL - An acronym for GNU General Public License, a license
  • GUI - An acronym for graphical user interfaces
  • Hardware description language - In electronics, a HDL (an acronym for hardware description language) is any language from a class of computer languages for formal description of electronic circuits
  • HCI - An acronym for Human-Computer Interaction
  • ILU - An acronym for Inter-Language Unification system, one of the most sophisticated Foreign Function Interface generators
  • IPC - An acronym for InterProcess Communication
  • IRC - It is an acronym for Internet Relay Chat
  • JIT - An acronym for Just In Time
  • LMNL - An acronym for Layered Markup and Annotation Language
  • LoI - LoI is an acronym for "Language of Interaction"
  • LtU - An acronym for Lambda, the Ultimate Programming Languages Weblog
  • MIA - An acronym for "Missing In Action"
  • MISC - An acronym for Minimal Instruction Set Computer
  • ML - An acronym for Meta Language (originally for the theorem prover LCF) is a class of non-lazy, statically-typed, statically-compiled, pattern-matching functional programming languages
  • MMU - An acronym for Memory Management Unit
  • Moostrap - An acronym for Mini Object-Oriented System Towards Reflective Architectures for Programming: a prototype-based object-oriented programming language with behavioral reflection
  • MOP - An acronym for Meta-Object Protocol
  • MST - An acronym for Meta-System Transition theory
  • MUD - An acronym for Multi User Dungeon
  • MVC - An acronym for the Model-View-Controller user interface architecture
  • NASM - NASM is an acronym for the Netwide ASseMbler, an assembler for Intel x86 assembly
  • NIAL - An acronym for Nested Interactive Array Language, a collection-oriented programming language
  • OO - An acronym for Object-Oriented
  • OOP - An acronym for Object-Oriented Programming
  • OS - The topic OS is an acronym for Operating System
  • PD - An acronym for Public Domain
  • PDF - PDF is an acronym for Portable Documentation Format, by Adobe System, Inc.
  • Pilot - Acronym for Programmed Inquiry, Learning, Or Teaching language
  • PJ - An acronym for Pictorial Janus, a visual concurrent constraints programming language based on Janus
  • programmable logic device - A PLD (acronym for programmable logic device) is an electronic component used to build digital circuits
  • ps - PS is an acronym for PostScript
  • PVM - An acronym for Parallel Virtual Machine, a concrete VM
  • QoS - An acronym for Quality of Service
  • REPL - An acronym for the Lisp programming language family's very general and famous meta-level architecture for extension
  • RI - An acronym for ResearchIndex
  • RISC - An acronym for Reduced Instruction Set Computer, a kind of CPU
  • RTOS - An acronym for Real-Time OS
  • Salsa - An acronym for Simple Actor Language and System Architecture: a distributed actor-based programming language with syntax similar to Java's, while also running on the JVM platform
  • SAS - An acronym for Single Address Space
  • SCP - SCP is an acronym for Supervised Compilation or Supercompilation
  • SEI - An acronym for Software Engineering Institute at Carnegie Mellon University
  • SGML - SGML is an acronym for Structured Generalized Markup Language
  • SICP - SICP is an acronym for "Structure and Interpretation of Computer Programs", a book written by Harold Abelson, Gerald Jay Sussman and Julie Sussman, published by MIT press and used as a textbook at MIT for introductory courses
  • TAL - An acronym for Typed Assembly Language, which extends traditional untyped assembly programming languages with typing annotations, memory management primitives, and a sound set of typing rules
  • THE - THE an acronym for The Humane Environment
  • TIGRE - An acronym for Threaded Interpretive Graph Reduction Engine, a concrete VM described in Philip J. Koopman, Jr. Ph
  • Tps - An acronym for Tiny/Transportable Postscript, a Postscript-like programming language with all of the graphics operators removed and Heterogeneous State Transportability: the ability to interrupt-migrate-resume a computation (even on a different machine architecture/OS) conceived to programming Agents
  • TUI - An acronym for both Text and Textual User Interface
  • TUNES HLL - The TUNES HLL is an acronym for "High-Level Language" or perhaps "High-Level Library", since the notion of language in Tunes is supposed to be fluid
  • TUNES LLL - LLL is an acronym for "Low-Level Language" or perhaps "Low-Level Library", since the notion of portability language in Tunes is supposed to be fluid
  • UI - An acronym for User Interface
  • UTM - An acronym for Universal Turing Machine, a machine capable of emulating any other discrete-state machine
  • VB - An acronym for Visual BASIC
  • VM - An acronym for Virtual Machine
  • WIMP - An acronym for Windows, Icons, Mice, and Pull-down menus
  • WYSIWYG - A User Interface related acronym for What You See Is What You Get
  • WYSIWYM - An acronym for the user interface concept of "What you see is what you mean
  • XML - An acronym for eXtensible Markup Language
  • ZUI - An acronym for Zoomable User Interface
