Reference

The term for an abstraction of the use of an object.

There are various interpretations and levels of abstraction for this concept, including low-level addresses, late-bound addresses (keyed-resolution), and symbolic (lexical) references.

Linear naming refers (sic) to a type of a reference allowed.

In the more common uses of the term, References are "considered harmful"; how objects are or can be used should be well-mannered or finely-controllable.
