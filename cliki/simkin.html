Simkin

Simkin is a simple interpreted programming language that can be placed within data files, including XML or within databases.

It can used to enable users to customize a C++ or Java-based application using scripts.

The language is interpreted by the C++ or Java classes provided in the package, which provide direct support for embedding the scripts within XML documents or within "TreeNode" files.
