User Interfaces

User Interfaces reviewed and related terms.

  • Nooface In Search of the Post-PC Interface, a user interface cooperative news site.
  • Ask Tog, rants and raves from Bruce Tognazzini.
  • The Anti-Mac Interface, by Don Gentner and Jakob Nielsen: By exploring alternative interfaces that transcend the principles behind conventional graphical interfaces, a human-computer interface emerges that is based on language, a richer representation of objects, expert users, and shared control.
  • Towards the Anti-Mac.

  • Agent - A user interface term for a unit of migratable control that is often dedicated to a specific domain or class of tasks
  • Animation - A user interface term for sequential display of graphics, intended to capitalize on the human visual ability to provide an understanding of the collection in a continuous form
  • Archy - It was The Humane Environment (THE), is a very well thought out Textual and Zoomable User Interface (TUI & ZUI), implemented in Python
  • CLI - An acronym for Command Line Interface, a kind of user interface
  • CLIM - CLIM is an acronym for Common Lisp Interface Manager
  • Diagram - A user interface term for a representation of some group of information that at least partially makes use of structural or symbolic representation
  • DWIM - An acronym for a user interface strategy called "Do What I Mean", where common programming errors are corrected in a live sense
  • Eidola - From its home page (see below):
  • Garnet - A User Interface toolkit for Common Lisp
  • GOMS - A user interface concept, acronym for Goals, Operators, Methods and Selection rules, a model of efficiency in HCI
  • GUI - An acronym for graphical user interfaces
  • Haystack - A user interface and an Information Retrieval system for the average computer user
  • Hypertext - A user interface term coined by Ted Nelson to discuss text as a medium extended to support any kinds of internalized references to other pieces of text
  • LAPIS - A user interface, a programmable web browser and text editor that demonstrates how lightweight structure can be useful
  • LifeStreams - A user interface term: it is a stream of information chunks to organize the way a person deals with other people; in much the same manner as an electronic diary, the chronological arrangement is key, but the information can be organized in many other ways
  • Memex - A set of devices and concepts associated with the essay, As We May Think, from early computing history, which constitute some original visions about user interfaces:
  • Modal - Modal is a user interface term, but it is applied from the field of logic, where a mode means a situation where only a finite set of possibilities for transitioning into the future are possible
  • Model-View-Controller - A user interface architecture originating in the 1970's in the Xerox Smalltalk system, where three objects play roles that combine into proper elements on the screen:
  • Morphic - The user interface system introduced as an integrated part of the Self system, and later ported to Squeak
  • OLIVE - A collection of resources related to information visualization (HCI, User Interface)
  • On Automatic Interface Generation - See Tunes Interfaces Subproject -- Interface Generation
  • Terminal - The user interface term which, for the purposes of Tunes, is the combination of any output device and any input device
  • Time-Machine Computing - A user interface term for a time-centric approach to organizing information on computers; it differs from LifeStreams in the use of both time and spatial (2D desktop) dimensions (where for LifeStreams the spatial dimension is not meaningful, it is automatically managed)
  • TUI - An acronym for both Text and Textual User Interface
  • Visible - Visible is a User Interface term
  • Vital - A TUI-based (document-centered user interface) implementation of Haskell which features: document-centered presentation of Haskell programs and results, demand-driven evaluation of expressions, diagrammatic representation of data structures and ADTs using styles, graphics (including animation and interaction), direct manipulation of data structures and of abstract datatype values, persistence
  • Wily - Wily is a TUI style of user interface for Unix-like OSes inspired by the Plan9 Acme TUI
  • WIMP - An acronym for Windows, Icons, Mice, and Pull-down menus
  • windowing system - Frankly, I did not even read any of these user interfaces links
  • WYSIWYG - A User Interface related acronym for What You See Is What You Get
  • WYSIWYM - An acronym for the user interface concept of "What you see is what you mean
  • Zero Memory Widget - A user interface resource, a widget library that uses zero bytes of memory per widget
  • ZUI - An acronym for Zoomable User Interface
