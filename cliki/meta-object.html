Meta-Object

The term for an object whose sole focus of attention is another object. It describes some feature or aspect of it.

There is a specialized use of this term, which may be characterized as describing "universal" meta-objects, wherein the entire set of objects that describe the original object in its existence as such are collected or represented by a single object encompassing all those aspects.
