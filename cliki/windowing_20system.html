windowing system

Frankly, I did not even read any of these user interfaces links. Anyway, what I call windowing system, is the way to move, organize and navigate through the windows. This if for me a part of the user interface, but to some point, not the point we usually focus in.

So I created this node to speaks about a simple windows navigation model, I'd like to see more used, let's called it hierarchical windows navigation model. -- Paul Dufresne

Even for PCs and workstations, a WIMP is not the ultimate user interface model. Please, have a look at some nodes in User Interfaces. -- MaD70
