I am the person who now maintains RetroForth
RetroForth is a public domain Forth system for DOS, Windows, Linux, or standalone use. I am working on both the hosted and native aspects to try to get a usable system written.

My vision for Retro: A compact programmer's environment programmable in Forth and Assembly. The user interface would be mainly text based, though some simple graphics would exist. It would be fully keyboard controlled. The applications would be a small block editor, abc player, 2D CAD, serial terminal, and email client.
