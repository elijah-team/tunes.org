AdaOS

A project to create a new fully distributed OS, written mostly in Ada.

Features will be: microkernel based; highly concurrent; object-oriented infrastructure, with reflection; support for a GUI; realistic security facilities.

The objective is mainly to demonstrate Ada as a systems programming language.

Current status: early stages; very gradual progress.

License: GPL/LGPL/GFDL (with minor modifications).
