Agnix

Agnix is a small OS kernel for i386, supporting 32-bit protected memory mode, paging, hardware switched tasks, memory tests (RW, ECC, address patterns), PCI bus, devices, PCI IRQ routing, RT timers and network protocols. Agnix is available with all the source code and is fully compatible with the Linux Kernel API. Agnix has been written from scratch by Lukas Dembinski.
