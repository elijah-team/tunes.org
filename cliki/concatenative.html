concatenative

The term for a sub-paradigm of programming in which function composition is rendered syntactically as concatenation (composition, juxtaposition) of symbols by means of a composition operator, usually rendered as a blank space.

  • concatenative Yahoo! group.
  • Concatenative Languages Wiki, by Slava Pestov (see also Factor).

A list of concatenative (or mostly concatenative) languages:

  • Factor - A concatenative programming language
  • Forth - A low-level functional programming language, mostly concatenative, with imperative features, for a stack-based VM model, invented by Chuck Moore
  • XY - A concatenative programming language
