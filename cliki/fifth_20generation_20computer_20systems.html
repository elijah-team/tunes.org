Fifth Generation Computer Systems

(FGCS) was a japanese project for a new generation of computer systems obtained by combination of Knowledge Information Processing and Large-scale Parallel Processing using parallel logic programming paradigm (Concurrent Prolog, KL1, ...).

  • Fifth Generation Computer Systems Museum, a repository of research reports, proceedings, notes, software, housed..
  • .. at Institute for New Generation Computer Technology (ICOT).

  • ICOT Free Software a large repository of software, the outcome of this research project.
  • The KLIC Association aims at development and popularization of KLIC, an implementation of a concurrent and parallel logic programming language KL1, and related software.
