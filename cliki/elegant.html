Elegant

An acronym for Exploiting Lazy Evaluation for the Grammar Attributes of Non-Terminals, a full imperative programming language by Philips Research, which has been inspired by abstraction mechanisms found in modern functional languages, started as a compiler generator based on attributed grammars. Release 7 is distributed under the GNU GPL.
