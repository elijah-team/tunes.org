Alliance OS

Was a split from Freedows, its goal was to implement an OS based on isolated components, described with the Corba IDL and communicating through a System ORB. This project was very organized (I'd say, lots of bureaucracy) but documentation and code were still sparse.

  • Alliance OS MIA. When I tried to get to the website (03/05/03), I got a "For Sale" sign. Ditto (03/11/02). Don't browse Alliance OS with IE, unless you want to lose control of your machine. -- A N Other.
