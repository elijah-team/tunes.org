Mica

Formerly known as Poe/Cool++, a rewrite/refactoring/revisiting in C++ of the persistent-object shared-programmer MOO-like system CoolMUD.

An interpreter for a garbage collected, dynamically typed, OO programming language with prototype-based inheritance, multiple dispatch, strict encapsulation and capabilities for security, optional orthogonal persistence, and functional programming features (anonymous functions, functional lists and dictionaries, map and filter operators).
